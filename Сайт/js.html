Основные теги JS

1. Динамическое выполнение кода

JavaScript позволяет динамически изменять содержимое страницы без перезагрузки. Это достигается за счет манипуляций с DOM (Document Object Model), который представляет собой структуру документа HTML. Например, можно добавлять новые элементы, скрывать существующие, менять стили и т.д.

Пример изменения текста заголовка:

document.getElementById('header').innerText = 'Новый заголовок';

2. Асинхронность

JavaScript поддерживает асинхронное программирование, что особенно полезно при выполнении длительных операций, таких как запросы к серверу. Это позволяет избежать блокировки основного потока выполнения программы.

Пример использования fetch для получения данных с сервера:

fetch('https://example.com/api/data')
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Ошибка:', error));

3. Функции высшего порядка

Функции в JavaScript являются объектами первого класса, что означает, что они могут быть переданы как аргументы другим функциям, возвращены из функций и присвоены переменным. Это открывает возможности для функционального программирования.

Пример передачи функции в другую функцию:

function greet(name, callback) { callback(`Привет, ${name}!`); }

greet('Мир', alert); // Выведет окно с сообщением "Привет, Мир!"

4. Объектно-ориентированное программирование

Хотя JavaScript изначально был прототипным языком, он также поддерживает объектно-ориентированный стиль программирования благодаря классам, добавленным в ES6.

Пример определения класса:

class User {
constructor(name, age) {
this.name = name;
this.age = age;
}
sayHello() {
console.log(`Привет! Меня зовут ${this.name}, мне ${this.age} лет.`);
}
}
const user = new User('Иван', 30);
user.sayHello(); // Привет! Меня зовут Иван, мне 30 лет.

5. Модули

Модульная система в JavaScript позволяет разбивать код на отдельные модули, каждый из которых может экспортировать и импортировать определенные части своего функционала. Это помогает организовать код и управлять зависимостями.

Пример экспорта и импорта модуля:

// file.js
export function add(a, b) {
return a + b;
}
// main.js
import { add } from './file.js';
console.log(add(2, 3)); // 5

6. Управление событиями

JavaScript предоставляет мощные механизмы для управления событиями, такими как клики мыши, прокрутка страницы, загрузка документов и многое другое. События позволяют реагировать на действия пользователя и выполнять соответствующие операции.

Пример добавления обработчика события на клик:

document.getElementById('myButton').addEventListener('click', function() {
alert('Кнопка была нажата!');
});

7. Работа с JSON

JSON (JavaScript Object Notation) — это формат обмена данными, основанный на синтаксисе объектов JavaScript. JavaScript предоставляет встроенные методы для преобразования данных в JSON и обратно.

Пример преобразования объекта в строку JSON:

const user = { name: 'Иван', age: 30 };
const jsonString = JSON.stringify(user);
console.log(jsonString); // {"name":"Иван","age":30}

8. Поддержка современных стандартов

JavaScript постоянно развивается, и современные стандарты, такие как ECMAScript 2015 (ES6) и последующие версии, добавляют новые возможности и улучшения. Это включает стрелочные функции, деструктуризацию, классы, промисы и многое другое.

Пример использования стрелочной функции:

const numbers = [1, 2, 3, 4, 5];
const doubledNumbers = numbers.map(num => num * 2);
console.log(doubledNumbers); // [2, 4, 6, 8, 10]
